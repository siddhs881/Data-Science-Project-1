# Data-Science-Project-1

**Author:** Siddhant Shah
**Course:** CMP262 - Data Science Programming

**Project Purpose:** The purpose of this project is to analyze survey data from computing students at the County College of Morris (CCM) from the Fall 2024 semester.
By performing data cleaning and exploratory analysis, this project aims to provide the CCM IT Department with key data analysis regarding student demographics, the
effectiveness of different recruitment tactics, and the main motivations the cause students to enroll in computer science/IT courses.

**Repository Files:** 
- **Majors Survey Results - Fall 2024.csv:** The original data source provided.
- **cleaned_majors_survey_results.csv:** The final cleaned dataset generated after handling missing values and renaming columns.
- **datacleaningexploration.ipynb:** The first part of the project, focused on data tidying and simpifying survey headers.
- **dsanalysis.ipynb:** The final analysis notebook containing visualizations and findings for the data study questions.
